# iedit.vim

iedit mode deteched from SpaceVim

## Installation

Use your preferred Neovim plugin manager to install mru.nvim.

with [nvim-plug](https://github.com/wsdjeg/nvim-plug)

```lua
require('plug').add({
    { 'wsdjeg/iedit.nvim' }
})
```

Then use `:PlugInstall iedit.nvim` to install this plugin.
